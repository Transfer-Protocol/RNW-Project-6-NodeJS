# Admin User Management <img src="./README_ASSETS/images/person-gear.svg">

## 🔹 Project Description

This is an Admin Panel that allows you to manage User Account using Web UI. It is build on top of Express.js and MongoDB. It uses cookies to
store user ID which is then being used to handle user login session. This cookie is stored for 24 hours after each login. You can also register your account for yourself or add new account if you already have admin access.

---

## 🔹 What is included in this project

- Admin Account Registration
- Login Page
- CRUD
- Profile Management
- Multer integration (profile picture)
- MongoDB integration
- Mobile Friendly
